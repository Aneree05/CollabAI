# CollabAI

An AI-powered freelance collaboration platform that enables clients and freelancers to collaborate efficiently through project management, real-time communication, file sharing, and AI-assisted recommendations.

## Features (Planned)

- User authentication and authorization
- Client and freelancer dashboards
- Project posting and bidding
- Real-time chat
- Task and milestone management
- File sharing
- AI-powered project recommendations
- AI cost estimation
- Notifications

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools
- Socket.IO
- JWT Authentication
- Cloudinary (for file storage)
